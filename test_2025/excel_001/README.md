# excel-001 项目文档

## 项目概览

本项目是一个数据处理和分析工具，用于Excel数据的读取、处理、分析和预测。项目采用Python开发，使用polars库进行高效的数据处理。

## 技术栈

- **编程语言**: Python 3.13+
- **核心依赖**: polars 1.35.2+

## 项目结构

- `main.py`: 主程序入口，包含数据处理示例代码
- `pyproject.toml`: 项目配置文件，定义依赖和元数据
- `README.md`: 项目文档
